# emoji

단어와 가장 유사한 이모지를 검색

## 예시

```
User: 🚰 물 + 🔥 불
                         = 🔌 전기
User: 🎐 바람 + 🎐 바람
                         = 🏙️ 풍경
User: 🚰 물 + 🔥 불
                         = 🔥 화재
User: 🏙️ 풍경 + 🔥 화재
                         = 🎆 자연불
User: 🔥 화재 + 🎐 바람
                         = 🌪️ 폭풍화재
User: 🔥 불 + 🎆 자연불
                         = 🍶 숯불
```
